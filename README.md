# karaoke-web-typist

A karaoke app featuring a web interface where users can input text and set duration for the text to be displayed. The text color changes as it is displayed, with special handling for text enclosed in '>' and '<' symbols to change color at 1.5x speed.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/karaoke-web-typist.git
cd karaoke-web-typist
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
